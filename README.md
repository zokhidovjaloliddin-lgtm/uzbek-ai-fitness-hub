### 📝 Project Overview
Absolute Frame is a high-intensity fitness application built with React.js. It features a unique "Warrior Briefing" gate that requires users to engage with motivational content before unlocking their personalized training protocols. 

### 🛠 Tech Stack
*   **Frontend:** React.js (Vite)
*   **Styling:** Tailwind CSS (Ultra-Noir Theme)
*   **Persistence:** LocalStorage (for subscription and state tracking)
*   **Deployment:** Netlify/Vercel via Lovable

### 🚀 Key Features
*   **Tiered Access:** Fully functional Free, Pro, and Ultra subscription logic.
*   **Cultural Localization:** Localized in Uzbek with an aggressive, direct tone ("Sen") to drive user engagement.
*   **Character Archetypes:** Integrated video training briefings for Yujiro Hanma, Kratos, Khabib, and Khamzat.
*   **Responsive Design:** Optimized for mobile training in any environment.
